Jubilee Austen Project
HTML 5 & CSS Website

This was my first project working with Photoshop and creating a webpage from a picture. This project consisted of grids, sections, unordered list, relative and absolute links. Styling information came from Photoshop to obtain the grid widths, colors, and text informtation from to style the webpage according to the photo. Floating was a new skill learned on this amazing project.

Rogue Pickings Website
HTML 5 & CSS from Photoshop Design Comp

My next project took it one step farther by adding borders, page breaks, and google fonts. Photoshop was used to extract information from the given photo to create this one page webpage. The outcome is amazing and many new skills were learned on this project.

Vision Board
HTML 5 & CSS Exercise

My skills were tested with this exercise and even picked up a few more. The HTML structure was a little different than other projects since it contained many divs, classes, and id's. On the styling side there is one main background and in some of the blocks the background was changed to a solid color for better effects. Clearing floats and block sizing became a breeze after this project and the styling is eye catching!
